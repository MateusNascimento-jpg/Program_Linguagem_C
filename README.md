# Program_Linguagem_C
Este é o meu primeiro repositório em Linguagem C, desenvolvido no primeiro semestre da faculdade na disciplina de Lógica de Programação. Ele reúne aproximadamente 20 exercícios focados nos conceitos básicos da linguagem, como variáveis, operadores, estruturas de decisão, laços de repetição e arrays. O objetivo é praticar a programação estruturada, desenvolver o raciocínio lógico e registrar minha evolução inicial na programação, servindo como base para estudos mais avançados ao longo do curso.
