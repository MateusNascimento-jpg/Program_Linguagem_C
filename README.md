# Lógica de Programação | Linguagem C

Este repositório consolida a base fundamental em desenvolvimento de software através da Linguagem C. O foco principal consistiu no domínio da lógica de programação e da estruturação de algoritmos, estabelecendo os alicerces necessários para a construção de sistemas complexos no escopo da Engenharia da Computação.

## O que foi desenvolvido?
O projeto reúne uma bateria de aproximadamente 20 desafios práticos e progressivos. A abordagem computacional abrangeu desde a sintaxe básica até a resolução de problemas algorítmicos, com ênfase estrita na programação estruturada e na execução sequencial e lógica de instruções.

## Tecnologias e Conceitos Aplicados
O desenvolvimento priorizou a construção de um pensamento analítico focado na eficiência e na correta manipulação dos recursos da linguagem. As principais competências trabalhadas incluem:

* **Programação Estruturada:** Implementação rigorosa de algoritmos utilizando estruturas de controle de fluxo, abrangendo tanto a tomada de decisão (condicionais) quanto a iteração (laços de repetição).
* **Gestão de Dados Primitivos:** Definição, alocação e manipulação de variáveis, aplicando corretamente os tipos de dados e os operadores lógicos e relacionais da linguagem C.
* **Estruturas de Dados Estáticas:** Utilização avançada de arrays para o armazenamento, indexação e processamento sequencial de blocos de informação na memória.
* **Raciocínio Computacional:** Decomposição de problemas de engenharia em passos lógicos executáveis, criando a base metodológica para o futuro desenvolvimento de arquiteturas de software mais robustas.
